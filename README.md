# 🚀 Hyper-Vulkan Dynamic Switcher

Optimización inteligente de renderizado para dispositivos Xiaomi (HyperOS/MIUI) con Root.

## 🛠️ ¿Qué hace este Mod?
Este módulo soluciona el conflicto histórico entre la estabilidad de las apps diarias y el rendimiento en juegos pesados.

* **Modo Diario (OpenGL/SkiaGL):** Mantiene el sistema estable para evitar errores en videos de Facebook, YouTube y la Grabadora de Pantalla.
* **Modo Gaming (Vulkan/SkiaVK):** Detecta automáticamente cuando abres un juego pesado y activa el motor Vulkan para maximizar los FPS.

## 🎮 Juegos Soportados (Auto-Detection)
El script monitoriza en tiempo real más de 14 títulos, incluyendo:
* Subnautica: Below Zero
* Call of Duty: Shooter
* Genshin Impact
* Warframe Mobile
* GTA Series (SA, LCS)
* Truckers of Europe 3
* ¡Y muchos más!

V2
HyperVulkan Switcher v2.0 - Dynamic Optimization Engine
Lleva el rendimiento de tu dispositivo al siguiente nivel con la nueva versión estable de HyperVulkan Switcher. Este módulo Magisk no es solo un simple set de props; es un motor inteligente que detecta en tiempo real qué estás haciendo con tu móvil para ofrecerte la máxima potencia en juegos y el mejor ahorro de energía en el uso diario.  
¿Qué hace a la v2.0 especial?
Auto-Switcher Inteligente: El servicio monitoriza la ventana activa y conmuta automáticamente entre el renderizado Skia Vulkan (para juegos) y Skia OpenGL (para apps normales), optimizando el consumo de batería.
Compatibilidad Universal: Diseñado originalmente para Snapdragon (Adreno), pero ahora con soporte optimizado para MediaTek y Exynos mediante ajustes configurables en el script base.
Gaming Pro: Integración con MIGT (Xiaomi Game Turbo) y optimizaciones específicas para motores Unity, mejorando la estabilidad de los FPS y la respuesta táctil.
Sistema Blindado: Incluye un fix avanzado de Integrity/VBMeta que simula un estado de arranque verificado para mejorar la compatibilidad con apps bancarias y juegos con anti-cheat.
Audio Hi-Fi & Bluetooth: Optimización del buffer de audio y soporte para LDAC/AAC, garantizando un sonido premium sin retrasos en juegos.
## 📦 Instalación
1.  Descarga el repositorio (o el .zip en Releases).
2.  Instala a través de **Magisk**, **Kitsune Mask** o **KernelSU**.
3.  Reinicia y disfruta.
